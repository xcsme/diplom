# Кофейня: Мастерская вкуса

Симулятор управления кофейней — игра, в которой вы управляете собственным бизнесом.

## Описание

Вы — владелец небольшой кофейни. Закупайте ингредиенты, устанавливайте цены, покупайте улучшения и стремитесь к успеху!

### Механика игры

- **Закупка ингредиентов** — покупайте кофейные зёрна, молоко, сахар, сиропы и другие ингредиенты
- **Управление меню** — устанавливайте цены и включайте/выключайте позиции
- **Улучшения** — покупайте оборудование и нанимайте персонал для развития бизнеса
- **Случайные события** — каждый день могут произойти различные события, влияющие на игру
- **Достижения** — отслеживайте свой прогресс и получайте достижения

### Условия победы и поражения

- **Победа**: накопите 100 000 ₽
- **Поражение**: деньги или репутация падают до 0

## Запуск

### Бэкенд

```bash
cd backend
pip install -r requirements.txt
python server.py
```

### Фронтенд

```bash
cd frontend
npm install
npm start
```

## Технологии

- **Backend**: Python, FastAPI, MongoDB
- **Frontend**: React, Tailwind CSS, Recharts
